Must-Have Features for v1
These are non-negotiable. If these don't work, the app is worthless.
1. Voice Conversation Interface

Start a session with one tap
Talk naturally without thinking about structure
AI responds with voice, asks follow-up questions
Can pause/resume during session
Session ends when I'm done talking

No text interface required for v1. Voice-only is fine.
2. Conversational AI That Extracts Detail
The AI must:

Listen and acknowledge what I'm saying
Ask clarifying questions when something is vague
Probe deeper on important topics (decisions, emotions, relationships)
Connect current topics to past conversations when relevant
Know when to shut up and just let me talk

This requires good prompting. The AI should feel like a thoughtful friend, not an interviewer.
3. Transcript Storage
Every conversation gets:

Full transcript saved
Timestamp
Basic metadata (duration, word count)

Raw transcripts are the source of truth. Everything else is derived from them.
4. Memory Synthesis
After each session:

AI generates a structured summary of key points
Extracts: decisions made, emotional states, important events, people mentioned
Tags topics/themes
This becomes a queryable "memory node"

These summaries need to be good. They're what enables retrieval.
5. Semantic Search & Retrieval
The AI must be able to:

Find relevant past conversations based on current topic
Surface similar emotional states from the past
Reference past decisions when I'm facing similar situations
Do this through vector similarity, not just keyword matching

When I talk about feeling anxious about a project, it should recall the last time I felt that way and what happened.
6. Context-Aware Follow-Up Questions
During a conversation, the AI should:

Reference specific past entries naturally ("Last time you mentioned X, how did that turn out?")
Ask questions that build on historical patterns ("You usually feel this way when Y is happening. Is that true now?")
Use past context to ask deeper questions faster

This is what makes the app valuable over time. Early sessions are basic. Later sessions are insightful because the AI knows me.
What Can Wait for v2

Text interface for reviewing past entries
Manual editing of transcripts
Export functionality
Multiple conversation templates
Reminders/notifications
Mood visualization
Search UI for browsing history

These are nice-to-have. They don't affect whether the core loop works.
The Core Loop That Must Work

I open the app
I talk about what's on my mind
AI asks good questions that extract useful detail
Conversation ends, transcript saved
Memory synthesized and indexed
Next time I talk, AI recalls relevant context
This makes the next conversation better

If this loop works, everything else is polish. If this loop doesn't work, nothing else matters.
